# Social-Media-Analytics-Excel
## Project Overview
This project focuses on analyzing social media and engagement metrics using a Spotify dataset to understand audience behavior and brand performance. The analysis was conducted using Microsoft Excel to clean, organize, and interpret data for strategic branding insights.

## Tools Used
- MS-Excel
- Pivot Tables
- Excel Functions & Formulas
- Data Visualization (charts)

## Key Insights
- Identified content types with higher engagement levels
- Observed trends in audience interaction across time
- Highlighted metrics useful for brand visibility evaluation
